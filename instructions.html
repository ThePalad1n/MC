Installation

1) Download Files
2) Install Forge
3) Add Mods(PC/MAC)
4) Find Server
5) Feedback

Download & Extract

Before you continue down the page make sure that you have gone to the 'resources' page at the top right of the page and downloaded your forge and both mod packs. Put the downloads in an easily accessible place because you will be going to them several times.


Once you have downloaded them make sure you extract them in the same spot for easy access. Then make a new folder called 'mods' no capitals. Combine the contents of 'modspt1' and 'modspt2' in your mods folder and then proceed to the next step.

Installing Forge

Select the folder labeled 'Forge'. Next you will want to double click and run the installation. Please choose the client option and click OK. Do not change the file path, it will default to the correct area.


Next you will want to run Minecraft. Before you hit play go to the top of the screen under 'Installations'. Click on the one labled 'Forge', go back to the 'Play' tab and hit PLAY at the bottom of the screen.


If you are left at the main menu of Minecraft you're good to go and can move one to the next step. If not then contact me and we will walk through it

Adding Mods (PC)

Open 'M0dPack 2.0' folder and copy the 'mods' folder. Now hit the windows key and type %appdata%, then hit enter. There should be a folder called .minecraft. Go into that folder and paste your mods folder. It will ask if you want to overwrite, hit YES.


Now run Minecraft Forge like in the previous step and you should get to the main page but now in the bottom left-hand corner it should say '43 mods loaded'.


If you do not get a successful lauch contact me. Else, move on to next step

Adding Mods (MAC)

Open 'M0dPack 2.0' folder and copy the 'mods' folder. On mac open finder, hold down ALT and click Go then Library in the top menu bar. Open the folder Application Support and look for Minecraft.


Now run Minecraft Forge like in the previous step and you should get to the main page but now in the bottom left-hand corner it should say '43 mods loaded'.


If you do not get a successful lauch contact me. Else, move on to next step

Finding Server

Now that you have forge running with all the mods we want to connect to the server. So lauch forge, go to the multiplayer tab on the main menu. Select driect connect and enter:

66.235.175.231:25565

Then hit connect. You should now be in the world.
