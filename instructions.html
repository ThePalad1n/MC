Installation

1) Download Files
2) Install Forge
3) Add Mods(PC/MAC)
4) Find Server
5) Feedback

Download & Extract

Before you continue down the page make sure that you have gone to the 'resources' page at the top right of the page and downloaded your forge and the mod pack. Put the downloads in an easily accessible place because you will be going to them several times.


Once you have downloaded them make sure you extract each of them in the same spot for easy access.

Installing Forge

Select the folder labeled 'Forge'. Next you will want to double click and run the installation. Please choose the client option and click OK. Do not change the file path, it will default to the correct area.


Next you will want to run Minecraft. Before you hit play go to the top of the screen under 'Installations'. Click on the one labled 'Forge', go back to the 'Play' tab and hit PLAY at the bottom of the screen.


If you are left at the main menu of Minecraft you're good to go and can move one to the next step. If not then contact me and we will walk through it.

Adding Mods (PC)

Copy the 'mods' folder you extracted in the first step. Now hit the windows key and type %appdata%, then hit enter. There should be a folder called .minecraft. Go into that folder and paste your mods folder. It will ask if you want to overwrite, hit YES.


Now run Minecraft Forge like in the previous step and you should get to the main page but now in the bottom left-hand corner it should say '54 mods loaded'.


If you do not get a successful lauch contact me. Else, move on to next step.

Adding Mods (MAC)

Copy the 'mods' folder you extracted in the first step. On mac open finder, hold down ALT and click Go then Library in the top menu bar. Open the folder Application Support and look for Minecraft.


Now run Minecraft Forge like in the previous step and you should get to the main page but now in the bottom left-hand corner it should say '54 mods loaded'.


If you do not get a successful lauch contact me. Else, move on to next step.

Finding Server

Now that you have forge running with all the mods we want to connect to the server. So lauch forge, go to the multiplayer tab on the main menu. Select driect connect and enter:

66.235.175.231:25565

Then hit connect. You should now be in the world.

Feed Back

Got any feed back or had any trouble in the installation? Make sure to send me a msg, via the contact page, regarding any trouble you experienced during this. Happy Crafting.

Mod Pack Updates

If a new patch is here and you need to add/delete mods make sure that you do the following:

Frist go to the 'resources' page at the top right of the page and download the "updatepack". The open the extracted folder open the readme.txt. See which mods are being added or deleted. Now go back to the extracted folder and copy the contents.


PC Users

Now hit the windows key and type %appdata%, then hit enter. There should be a folder called .minecraft. Go into that folder and then go into the mods folder. If your readme.txt did not have any deletions then you are all set. If there were deletions make sure that you delete what is specified.


MAC users

On mac open finder, hold down ALT and click Go then Library in the top menu bar. Open the folder Application Support and look for Minecraft. Go into that folder and then go into the mods folder. If your readme.txt did not have any deletions then you are all set. If there were deletions make sure that you delete what is specified.


Now run Minecraft Forge like in the previous step and you should get to the main page but now in the bottom left-hand corner it should say '54 mods loaded'.
