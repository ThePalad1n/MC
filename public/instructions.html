Installation

1) Download Files
2) Install Forge
3) Add Mods(PC/MAC)
4) Find Server
5) Feedback

Download & Extract

Before you continue down the page make sure that you have gone to the 'resources' page at the top right of the page and downloaded your forge and the mod pack. It will bring you to our google drive page just click the little download button on the top right of the page and save it in an easily accessible place because you will be going to them several times. After you download and save hit hit the back button and do the same for the mod pack. Then return here to continue.


Once you have downloaded them make sure you extract each of them in the same spot for easy access.

Installing Forge PC

Select the folder labeled 'Forge'. Next you will want to double click and run the installation. Please choose the client option and click OK. Do not change the file path, it will default to the correct area.

Next you will want to run Minecraft. Before you hit play go to the top of the screen under 'Installations'. Click on add new. Name it Forge, under version select the one labled release 1.14.4-forge1.14.4-28.2.11. Then click SAVE. Click on the one labled 'Forge', go back to the 'Play' tab and hit PLAY at the bottom of the screen.

If you are left at the main menu of Minecraft you're good to go and can move one to the next step. If not then contact me and we will walk through it.

Installing Forge MAC

Note: There is a known issue with inistiall install with MACs regarding an 'unidentified developer' If you experience this issue click here

Select the folder labeled 'Forge'. Next you will want to double click and run the installation. Please choose the client option and click OK. Do not change the file path, it will default to the correct area.

Next you will want to run Minecraft. Before you hit play go to the top of the screen under 'Launch Options'. Click on add new. Name it Forge, under version select the one labled release 1.14.4-forge1.14.4-28.2.11. Then click SAVE. Now go back to the news tab at the top, click the little arrow next to the huge PLAY button and select Forge. Finally hit play.

If you are left at the main menu of Minecraft you're good to go and can move one to the next step. If not then contact me and we will walk through it.

Adding Mods (PC)

Copy the 'mods' folder you extracted in the first step. Now hit the windows key and type %appdata%, then hit enter. There should be a folder called .minecraft. Go into that folder and paste your mods folder. It will ask if you want to overwrite, hit YES.

Now run Minecraft Forge like in the previous step and you should get to the main page but now in the bottom left-hand corner it should say '54 mods loaded'.

If you do not get a successful lauch contact me. Else, move on to next step.

Adding Mods (MAC)

Navigate to your extracted mods folder. Hold CTRL and click on the folder and select copy. Now go to Go on the top right of your MAC. Now hold down ALT and a secret option called Library should appear and click on it. Now that you are in there navigate to the minecraft folder. Open the folder and now CTRL click and paste your copied mods folder. If it asks if you would like to overwrite the current one choose yes.

Now run Minecraft (With the Forge profile selected), like in the previous step, and you should get to the main page but now in the bottom left-hand corner it should say '54' mods loaded'.

If you do not get a successful lauch contact me. Else, move on to next step.

Finding Server

Now that you have forge running with all the mods we want to connect to the server. So launch Minecraft with forge, go to the multiplayer tab on the main menu. Select add server, make the server name "Valuecraft" and under server address enter:

66.235.175.231:25565

Then hit done. A new world named Valuecraft should pop up wait until the green on the right hand side pops up. Now hover over the little picture and click the play button. You should now be in the world.

Feed Back

Got any feed back or had any trouble in the installation? Make sure to send me a message on the contact page, regarding any trouble you experienced during this. Happy Crafting.

Mod Pack Updates

If a new patch is here and you need to add/delete mods make sure that you do the following:

Frist go to the 'resources' page at the top right of the page and download the "updatepack". The open the extracted folder open the readme.txt. See which mods are being added or deleted. Now go back to the extracted folder and copy the contents.


PC Users

Now hit the windows key and type %appdata%, then hit enter. There should be a folder called .minecraft. Go into that folder and then go into the mods folder. If your readme.txt did not have any deletions then you are all set. If there were deletions make sure that you delete what is specified.


MAC users

On mac open finder, hold down ALT and click Go then Library in the top menu bar. Open the folder Application Support and look for Minecraft. Go into that folder and then go into the mods folder. If your readme.txt did not have any deletions then you are all set. If there were deletions make sure that you delete what is specified.


Now run Minecraft Forge like in the previous step and you should get to the main page but now in the bottom left-hand corner it should say '54 mods loaded'.
